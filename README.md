# [ Projeto Arquitetura Computadores ]

> Este projeto usa Java para simular a microarquitetura MIC-I, baseada no livro "Organização Estruturada de Computadores", do Andrew S. Tanenbaum. 

## Tecnologias ⌨️ :
* **javaFX**
* **CSS**

## 🚀 Download e Execução

Você pode baixar a versão compilada e pronta para rodar do simulador. O aplicativo já vem com a Máquina Virtual Java embutida, então **não é necessário ter o Java instalado** na sua máquina!

## Windows
1. Baixe o arquivo zip clicando no link abaixo.
2. Extraia a pasta e execute o arquivo `Simulador Mic-1.exe`.

[📥 Baixar Simulador Mic-1 para Windows](https://github.com/Costa-08/Projeto_Arq/releases/download/v1.0.0/versao_windows.zip)

### Linux
1. Baixe o arquivo zip clicando no link abaixo.
2. Extraia a pasta.
3. Abra o terminal na pasta `bin` e execute o arquivo binário nativo.

[🐧 Baixar Simulador Mic-1 para Linux](https://github.com/Costa-08/Projeto_Arq/releases/download/v1.0.0/versao_linux.zip)
